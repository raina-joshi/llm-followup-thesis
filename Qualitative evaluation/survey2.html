Instructions

You will be shown a query and a set of follow-up questions generated by different methods. Please evaluate each question on three criteria:

  • Fluency: Is the question clearly written and easy to understand? Rate this based only on the language quality, grammar, clarity, and naturalness.
  • Relevance: Is the question actually about what the query is asking? Rate this based on how directly the question addresses the main intent or ambiguity of the query. A relevant question targets what the query is specifically about, not just any loosely related aspect of the topic.
  • Usefulness: If this question were answered, would the answer help clarify or resolve the original query? Rate this based on how useful the answer would be in determining the correct response to the query.

Please rate each criterion on a scale of 1 (lowest) to 5 (highest).
